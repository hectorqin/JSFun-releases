## JSFun 是一个什么样的 App？

JSFun 是一个实现了 [Dora.js](https://github.com/Dorajs/docs) 核心功能的App，使用了ReactNative,Electron,NodeJS等技术栈，支持全平台使用。 [JSFun official document](https://jsfun.vercel.app)
